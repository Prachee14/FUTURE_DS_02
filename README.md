# Customer Retention & Churn Analysis Dashboard

## About the Project

The goal of this project was to analyze customer churn and retention patterns using the Telco Customer Churn dataset. Through this analysis, I explored customer behavior, identified factors that contribute to churn, and developed insights that can help businesses improve customer retention.

This project helped me strengthen my skills in data cleaning, data visualization, business analysis, and dashboard development using Power BI.

---

## Problem Statement

Customer churn is a major challenge for subscription-based businesses. Losing customers can directly impact revenue and business growth.

The objective of this analysis was to answer the following questions:

* Why are customers leaving?
* Which customer groups are most likely to churn?
* How long do customers stay with the company?
* What actions can help improve customer retention?

---

## Dataset Used

**Telco Customer Churn Dataset**

The dataset contains customer demographics, subscription information, service usage, payment details, tenure, and churn status.

### Key Columns

* Customer ID
* Gender
* Senior Citizen
* Contract Type
* Internet Service
* Payment Method
* Monthly Charges
* Total Charges
* Tenure
* Churn Status

---

## Data Cleaning

To ensure data quality, the following cleaning steps were performed in Power BI:

* Checked for missing values
* Removed records with blank TotalCharges values
* Verified data types
* Checked for duplicate records
* Validated data consistency

### Data Summary

* Original Records: 7,043
* Cleaned Records: 7,032
* Missing Records Removed: 11

---

## Key Insights

* The overall customer churn rate is **26.58%**.
* Month-to-month contracts have the highest number of customers and are more likely to experience churn.
* Fiber Optic is the most commonly used internet service.
* Customers who churn generally have higher monthly charges than retained customers.
* Customers with longer tenure show better retention patterns.

---

## Recommendations

Based on the analysis, the following recommendations can help reduce customer churn:

1. Encourage customers to switch from month-to-month plans to long-term contracts.
2. Provide retention offers to customers with higher monthly charges.
3. Improve onboarding and engagement for new customers.
4. Identify high-risk customer segments and target them with retention campaigns.
5. Use customer behavior data to create personalized experiences.
